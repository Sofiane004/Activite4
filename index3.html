Les composantes matérielles d'un réseau

  • Les cartes réseau

    Les cartes réseau, ou cartes d'interface réseau (NIC), sont des composants informatique qui permet à un ordinateur de se connecter à un réseau informatique, que ce soit Internet ou un réseau local. Elles sont installées à l'intérieur de l'ordinateur et connectées à une prise Ethernet ou Wi-Fi. Les cartes réseau jouent un rôle crucial dans la communication entre ordinateurs sur un réseau en permettant la transmission et la réception de données à l'aide de protocoles de communication tels que TCP/IP.

  • Le Switch

    Un commutateur, également appelé commutateur réseau, est un composant matériel qui permet de connecter plusieurs appareils tels que des ordinateurs, des imprimantes ou des serveurs sur un réseau local (LAN) et de les faire communiquer entre eux en utilisant des adresses MAC. Lorsqu'un appareil se connecte au switch, son adresse MAC est stockée dans une table de correspondance qui est utilisée pour envoyer les données uniquement à l'appareil destinataire plutôt qu'à tous les appareils du réseau. Les commutateurs existent en différentes tailles.

  • Le HUB

    Un hub, également appelé concentrateur, est un composant matériel utilisé pour connecter plusieurs appareils sur un réseau local (LAN). Contrairement à un switch, le hub ne dispose pas de la fonctionnalité de table de correspondance pour envoyer les données uniquement à l'appareil destinataire. Cela signifie que lorsqu'un appareil envoie des données sur le réseau, tous les appareils connectés les reçoivent, même si l'appareil destinataire est différent. Les hubs ont été remplacés par des switches, plus efficaces pour la gestion du trafic réseau et la communication rapide et sécurisée entre les appareils connectés. Cependant, les hubs sont encore utilisés dans des applications spécifiques ou dans des petits réseaux où la complexité d'un switch n'est pas nécessaire.

Le modèle OSI

Le modèle OSI (Open Systems Interconnection) est un modèle de référence pour la conception et la mise en œuvre de réseaux informatiques, divisant les communications en sept couches distinctes. Chaque couche est responsable de fonctions spécifiques liées à la communication de données entre les appareils de réseau :

Les sept couches du modèle OSI sont :

  • La couche physique
  • La couche liaison de données
  • La couche réseau
  • La couche transport
  • La couche session
  • La couche présentation
  • La couche application

Le modèle TCP/IP

Le modèle TCP/IP est un modèle de référence pour les communications de réseau, utilisé pour la conception et la mise en œuvre des réseaux informatiques modernes. Contrairement au modèle OSI, qui divise les communications en sept couches distinctes, le modèle TCP/IP divise les communications en quatre couches principales :

  • La couche liaison de données
  • La couche réseau
  • La couche transport
  • La couche application

Le modèle TCP/IP est le modèle de référence pour les protocoles de communication de réseau les plus largement utilisés, telque IP/TCP/UDP...

L'adressage

Les communications dans un réseau obéissent à des règles :

  • Adressage
  • Architexture
  • Protocoles

C’est quoi un adressage IP ?

L'adressage IP (Internet Protocol) est un système utilisé pour identifier chaque périphérique connecté à un réseau.Il y’a deux mode d’adressage IPV4 et IPV6 :

  • IPV4 : Elle utilise une adresse IP de 32 bits, représentée sous forme de quatre octets décimaux séparés par des points.

  • IPV6 : Une adresse IPv6 est un identifiant unique de 128 bits, écrit en notation hexadécimale et composée de 8 groupes de 16 bits.

L'adresse IPV4

Ce passage décrit la structure d'une adresse IPV4 et les différents types d'adresses qui existent dans le schéma d'adressage IPV4. qui sont divisés en quatre octets. La première partie de l'adresse représente le réseau, tandis que la seconde partie représente l'hôte. Un réseau est reconnu lorsque la valeur des bits dans la partie réseau est la même. Le nombre de bits dans la partie hôte détermine le nombre de machines pouvant exister sur ce réseau. Il existe quatre types d'adresses IPV4 : adresse réseau, adresse de diffusion, adresse hôte et masque réseau. Une adresse réseau a tous les bits de la partie hôte définis sur 0, tandis qu'une adresse de diffusion a tous les bits de la partie hôte définis sur 1. Une adresse hôte est attribuée aux machines d'un réseau entre les adresses réseau et de diffusion. Un masque de réseau a tous les bits de la partie réseau mis à 1.
