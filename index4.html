Introduction

  • Le World Wide Web (WWW)

    Le World Wide Web (WWW) est un système hypertexte public sur Internet qui permet de consulter des pages accessibles sur des sites.

  • Le Web

    Le Web est une des applications d'Internet, qui est un support de communication général.les pages Web sont reliées entre elles par des hyperliens, d'où l'image de la toile d'araignée. Le Web sert des ressources de différentes natures : texte, image, son, vidéo, contenu applicatif et interactif (comme l'achat en ligne ou les discussions).

  • Les URL

    Les URL (Uniform Resource Locator) sont utilisées pour identifier et accéder à ces ressources, et contiennent l'adresse du serveur et l'adresse de la page sur le serveur.

Aspects techniques du Web

Les trois mécanismes de base du Web sont l'URL, HTTP et HTML :

  • L'URL : permet d'accéder à un ensemble de ressources et peut faire appel au protocole DNS pour la localisation.

  • L'HTTP : est un protocole de niveau applicatif qui suit un paradigme client-serveur et ne garde pas de mémoire des transactions précédentes.

  • L'HTML : est un langage de description de pages Web qui utilise du texte, des images et autres objets et permet des liens hypermédias entre les pages. Il utilise également une programmation déclarative.

Page Web et site Web

  • Une page Web

    Une page web est un document multimédia qui peut contenir des images, du texte, des liens et de l'interactivité. Elle est décrite à l'aide d'un langage de description tel que HTML, XHTML ou HTML5 et est stockée dans un fichier HTML, Les pages web sont affichées par les navigateurs web et peuvent être localisées sur Internet grâce à une adresse URL. peuvent également contenir des liens hypertextes permettant d'accéder à d'autres pages.

  • Un site Web

    Un site web est un ensemble de pages web liées entre elles par des liens hypertextes, émanant d'une même entité et accessibles à partir d'une page d'accueil commune et d'une URL de base commune. Les sites web sont généralement localisés sur un même serveur et sont structurés sous forme d'arborescence de dossiers et de fichiers, permettant de définir des chemins relatifs.

Création d’une page Web HTML (Hyper Text Markup Language)

Un document HTML est un fichier texte construit en utilisant un éditeur de texte ou de pages web, suivant une syntax précise,structuré par des balises et interpréte visuellement dans un navigateur.

Ces 4 balises sont nécessaires pour créer un fichier HTML :

  • <html>...</html> : Indique le début et la fin du document.
  • <head>...</head> : Entete le document.
  • <title>...</title> : Titre de document.
  • <body>...</body> : Corps du document.
  • <hn>...</hn> : Titre de niveau n (0<n<7).
  • <p>...</p> : Paragraphe.
  • <a>...</a> : Lien.
  • <img> : Image.
  • <li>...</li> : Liste.
